#Types#
The VM supports the following types:

__Primitive types__
* Int: 32 bit integer.
* Float: 32 bits floating point.
* Bool: True or false.
* Char: 8 bits integer.
* Void: Only allowed in return type which indicates no return value.

__Reference types__
* Arrays: `Ref.Array[<Element type>]`.
* Structs: `Ref.Struct.<Struct name>`.
